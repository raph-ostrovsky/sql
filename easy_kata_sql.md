#### Easy SQL: Convert to Hexadecimal

```sql
SELECT TO_HEX(legs) legs, TO_HEX(arms) arms FROM monsters
```

#### SQL Basics: Simple WHERE and ORDER BY
```sql
SELECT * FROM people
WHERE age > 50
ORDER BY age DESC
```
